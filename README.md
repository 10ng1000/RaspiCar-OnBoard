# raspicar-onboard
大创项目，部署了openEuler的树莓派小车在树莓派上的模块。
初步实现了麦轮控制全向运动，圆周运动，避障；视频传输，控制指令传输；板载基于yolo的目标检测功能。
[Android App](https://github.com/10ng1000/raspicar-android)
